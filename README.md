# Ashok Pandit and Company

Website for Ashok Pandit & Co., a Chartered Accountancy firm based in Mumbai, India.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local development

```sh
npm install
npm run dev
```

The dev server runs at `http://localhost:8080`.

## Build

```sh
npm run build
npm run preview
```
